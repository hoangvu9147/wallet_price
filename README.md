# wallet_price

Wallet_price  is a small demo application. With simple features.

## Installation

Use the package cocapod 

```bash
pod install 

pod 'CryptoSwift', '~> 1.4.1'
pod 'Alamofire', '~> 4.8.1'
pod 'SwiftyJSON', '~> 4.0'
pod 'MBProgressHUD'
pod 'SDWebImage'
pod 'CryptoSwift', '~> 1.4.1'
pod 'DropDown'
pod 'DPOTPView'
pod 'SwiftyRSA'
 
```
